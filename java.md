### 二维数组

数组地址存在栈中，指向堆中的数组空间，空间内存放数组地址，指向新的数组空间，新的数组空间中存放内容

```java
int[][] arr = new int[][]{{1,2,3,4},{3,4},{4,5,6}};
int[][] arr = new int[3][2];
//长度
arr.length//3
arr[0].length//4
```
